## 🐍 کد نمونه پایتون

در اینجا یک نمونه کد ساده در پایتون آورده شده است که نشان می‌دهد چگونه می‌توان یک تابع ساده برای خوش‌آمدگویی نوشت:

```python
def greet(name: str) -> str:
    """
    یک تابع ساده برای خوش‌آمدگویی به کاربر.
    
    :param name: نام کاربر
    :return: پیام خوش‌آمدگویی
    """
    return f"سلام {name}!\nخوش آمدید به دنیای برنامه‌نویسی پایتون! 🐍"

# مثال استفاده از تابع
print(greet("امیرحسین"))
